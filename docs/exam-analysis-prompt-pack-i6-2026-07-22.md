# 시험분석 GPT Image 프롬프트 팩 I6

## 생성 결과

작업본과 I5 시퀀스로 다음 텍스트를 결정적으로 만든다.

1. 프로젝트 마스터 프롬프트
2. 슬라이드별 상세 생성 프롬프트
3. 생성 이미지별 수정 프롬프트
4. 생성 이미지별 QA 프롬프트
5. 전체 시리즈 스타일 수정 프롬프트

## 마스터 규칙

- 4:5, 1080×1350px, 한 번에 한 장
- 학교 스타일과 색상·타이포·카드·페이지 번호 일관성
- 입력된 학교명·문항 수·배점·범위·문항번호·교재·출처 외 사실 추측 금지
- 사용자가 GPT 대화에 첨부한 문제 crop·손풀이를 그대로 사용
- 수식·도형·그래프·손글씨 재작성 금지
- 자산이 첨부되지 않았으면 가짜 문제/풀이를 만들지 말고 중단

## 복사·내보내기

- 마스터 프롬프트 복사
- 준비된 슬라이드의 `마스터+상세`, 수정, QA 개별 복사
- 모든 슬라이드가 준비된 경우에만 전체 복사
- 검수·백업용 TXT와 구조화 JSON 다운로드

TXT/JSON은 현재 local draft를 내보내며 서버 저장 완료를 의미하지 않는다. 저장 여부는 I3 상태 배지로 별도 확인한다.

## 준비 판정

슬라이드가 `needs_input`이면 상세 생성 복사를 비활성화한다. 특히 주요문항의 문제 crop 파일명/ID와 손풀이 슬라이드의 검증 손풀이 파일명/ID가 없으면 생성 가능으로 표시하지 않는다. 파일명/ID 입력은 실제 GPT 대화 첨부를 대신하지 않으며, 첨부 여부는 I7 사람 Gate에서 확인한다.

## 호출 경계

이 기능은 프롬프트 텍스트만 생성한다. OpenAI/GPT Image API, 유료 AI, 이미지 생성, Canva, Storage 업로드를 호출하지 않는다.
