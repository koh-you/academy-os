# GPT Image용 시험분석 카드뉴스 소스 - 2026-07-09

## GPT 프로젝트 의도와 맥락

벤치마킹 블로그의 카드뉴스는 개별 도형/텍스트를 수작업으로 조립한 디자인이라기보다, GPT image 계열로 카드 전체를 한 번에 렌더링한 산출물에 가깝다. 따라서 Academy OS의 목표도 Canva에서 세부 요소를 먼저 조립하는 것이 아니라, `시험 특징 소스 -> 카드별 GPT image 프롬프트 -> 이미지 생성 -> 선생님 검수 -> 최종 카드뉴스/블로그/인스타 반영` 흐름으로 잡는다.

핵심은 디자인 복제가 아니라 구조 복제다. 원본에서 배울 것은 `학교명/고사명`, `시험 구조`, `난도 흐름`, `주요문항`, `다음 대비`가 한눈에 보이도록 카드 전체를 정보 단위로 묶는 방식이다. 실제 문장, 브랜드, 색 조합, 장식 요소는 그대로 쓰지 않고, `으뜸수학 고태영T`의 블루/화이트 톤과 차분한 내신 분석 스타일로 재해석한다.

GPT 프로젝트에는 아래 3개 레이어를 분리해 넣는다.

- `분석 원천`: Vision AI가 읽은 학교별 시험 특징과 키워드 사전
- `제작 규칙`: 10장 카드뉴스 구성, 브랜드 톤, 금지 규칙
- `카드별 입력`: 한 번에 1장씩 생성할 카드 역할, 메인 문구, 강조 키워드, 숫자/표 정보

선생님이 검수한 시험 사실이 원본이다. GPT image는 카드 이미지를 만드는 도구일 뿐, 문항번호/교재명/등급컷/시험범위 같은 사실을 새로 추측하면 안 된다.

## 어떤 소스를 넘길 것인가

GPT image에는 raw JSON 전체가 아니라 아래 4가지를 넘긴다.

1. `브랜드/디자인 규칙`
2. `선택한 학교의 시험 특징`
3. `이번 카드 1장의 역할과 문구`
4. `금지 규칙`

원본 Vision AI 결과는 아래에 있다.

- 전체 요약/키워드 사전: `docs/exam-analysis-blog-vision-all-posts-2026-07-09.md`
- GPT 프로젝트 모듈 구조: `docs/exam-analysis-gpt-image-project-module-2026-07-09.md`
- raw OCR/분류 JSON: `.codex-temp/benchmark-vision-results/<logNo>-latest.json`
- 전체 aggregate JSON: `.codex-temp/benchmark-vision-results/all-posts-latest.json`

GPT image에는 `.codex-temp` raw JSON을 통째로 붙이지 않는다. 필요한 학교의 요약과 카드별 짧은 문구만 붙인다.

## GPT Image 공통 디자인 소스

```text
브랜드:
- 으뜸수학 고태영T
- 전문적이고 차분한 수학 내신 분석 카드뉴스
- 블루/화이트 중심, 포인트 컬러는 1개만 사용
- 원본 블로그 카드뉴스를 복제하지 말고, 구조적 장점만 참고

캔버스:
- 세로 카드뉴스 1080x1350
- 모바일에서 읽히는 큰 제목
- 정보는 3~5개 블록으로 분리
- 숫자, 문항 수, 등급컷, 교재명은 배지나 표로 강조

디자인 규칙:
- 상단: 학교명/학년/고사명
- 중앙: 카드의 핵심 결론 1문장
- 하단: 핵심 키워드 2~4개 또는 다음 행동
- 표/리스트는 짧게
- 장식보다 정보 구조가 먼저
- 카드마다 역할이 명확해야 함: 표지, 시험 구조, 난도 흐름, 출제 원천, 주요문항, 다음 대비, CTA

금지:
- 원본 카드뉴스 색상/레이아웃/장식 그대로 복제 금지
- 수학의기술 브랜드명 사용 금지
- 이미지에 없는 시험 사실 추측 금지
- 너무 긴 문장 금지
- 작은 글씨로 빽빽하게 채우기 금지
```

## GPT Image에 넘길 카드 단위 프롬프트 템플릿

```text
아래 소스를 바탕으로 1080x1350 세로 카드뉴스 1장을 만들어줘.

[브랜드/스타일]
- 으뜸수학 고태영T
- 블루/화이트 중심
- 전문적이고 차분한 내신 분석 스타일
- 숫자와 키워드가 한눈에 보이는 카드뉴스
- 원본 블로그 카드뉴스는 복제하지 않고 구조만 참고

[학교 시험 특징]
- 학교/학년/고사: {학교명 학년 고사명}
- 출제 원천: {교과서/부교재/학습지/유사유형}
- 시험 구조: {문항 수/배점/등급컷}
- 단원 비중: {단원별 핵심}
- 난도 흐름: {초반/중반/후반}
- 학교별 특징: {시간관리/조건해석/계산량/그래프해석 등}
- 주요문항 패턴: {문항번호/개념/틀리는 지점}
- 다음 대비: {훈련 방향}

[이번 카드]
- 카드 번호: {1~10}
- 카드 역할: {표지/시험 구조/난도 흐름/출제 원천/주요문항/다음 대비/CTA}
- 메인 문구: "{짧은 한 문장}"
- 보조 문구: "{짧은 설명 1~2줄}"
- 강조 키워드: {키워드 2~4개}
- 반드시 넣을 숫자/표: {있으면 입력}

[레이아웃 지시]
- 상단에 학교명과 고사명을 작게 배치
- 중앙에 메인 문구를 가장 크게 배치
- 핵심 숫자/키워드는 카드형 배지 또는 간단한 표로 표시
- 하단에는 다음 카드로 이어지는 짧은 문구 배치
- 한글 텍스트가 잘리지 않게 여백 확보

[금지]
- 원본 문장 장문 복제 금지
- 원본 브랜드명 사용 금지
- 제공되지 않은 시험 사실 추가 금지
- 너무 많은 텍스트 금지
```
