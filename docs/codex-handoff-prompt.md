# Codex Handoff Prompt

새 대화세션 또는 집 컴퓨터 Codex에서 아래 프롬프트를 붙여넣으면 됩니다.

```text
이 프로젝트는 academy-os입니다.
먼저 AGENTS.md, docs/current-worklog.md, docs/home-codex-setup.md를 읽고 현재 작업 큐를 확인해주세요.

작업 방식:
1. 구현 전에 docs/current-worklog.md의 작업 큐를 갱신합니다.
2. 한 번에 하나의 작업만 진행합니다.
3. 구현 후 npm run build로 확인합니다.
4. 완료한 내용과 남은 작업을 docs/current-worklog.md에 반영합니다.
5. 필요한 경우 GitHub main에 commit/push합니다.

현재 우선 작업:
학사일정 캘린더에서 시험기간 막대는 유지하고, 수학시험 날짜만 위로 튀어나온 탭처럼 표시해야 합니다.
현재는 막대가 끊겨 보입니다.
```

## 짧은 다음 작업 프롬프트 템플릿

```text
[다음]
학사일정 캘린더:
시험기간 막대는 유지하고, 수학시험 날짜만 위로 튀어나온 탭처럼 표시.
현재는 막대가 끊겨 보임.
```

## 새 세션에 꼭 알려야 할 것

- 사용자는 `작업 큐 정리 -> 구현` 흐름을 원합니다.
- 구현 중 사용자가 추가 요구를 보내더라도, 충돌하지 않으면 현재 작업 큐에 넣고 계속 진행합니다.
- 같은 수정이 반복되면 먼저 원인과 남은 미해결 목록을 정리한 뒤 진행 여부를 확인합니다.
- UI는 사용자가 보낸 스크린샷에 맞추되, MVP 핵심 흐름을 해치지 않는 선에서 단순하게 유지합니다.
- 학원명 표시는 `으뜸수학 고태영T`입니다.
- 기존 `koh_you_math`는 내부 식별자나 URL에서만 사용할 수 있습니다.
