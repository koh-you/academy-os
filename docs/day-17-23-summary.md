# Day 17-23 Summary

## Goal

학생 화면의 세부 탭을 실제 MVP 화면처럼 분리하고, 밀린 숙제 이후의 후속조치 흐름을 Day 23 범위까지 목업으로 연결한다.

## Completed

- 학생 화면에 `오늘`, `등록`, `전체`, `커리큘럼`, `평가`, `마이 페이지` 탭을 실제 클릭 가능한 화면으로 구현했다.
- `등록` 탭에서 학생이 숙제를 직접 등록할 수 있게 했다. 저장은 브라우저 localStorage 목업이며 외부 서버 전송은 없다.
- `전체` 탭에서 등록된 숙제 목록, 상태 배지, 진행률, 수정 준비, 삭제 잠금 상태를 표시했다.
- `커리큘럼` 탭은 아직 설정 전 안내 상태로 두었다.
- `평가` 탭은 배정된 시험이 없을 때의 빈 상태를 만들었다.
- `마이 페이지` 탭에 통계, 활동 로그, 내 정보 하위 탭을 추가했다.
- Day 17-23 범위로 `후속조치 센터`를 추가했다.
- 밀린 숙제에서 `숙제보충` 과제를 생성하는 목업 흐름을 만들었다.
- 결석 기록에서 `결석 보강` 후보를 볼 수 있는 구조를 만들었다.
- 재시험 필요 기록에서 `재시험` 후보를 볼 수 있는 구조를 만들었다.
- 후속조치별 배정일, 시간, 상태, 알림 문구 초안을 편집할 수 있게 했다.
- 알림은 실제 발송하지 않고 `모의 로그`만 남기도록 했다.

## Guardrails

- 실제 삭제는 구현하지 않았다. 학생 숙제 목록의 삭제 버튼은 잠금 상태다.
- 실제 학부모/학생 알림 발송은 구현하지 않았다.
- 출결앱, AI API, Tally, BTLmath 외부 연동은 아직 연결하지 않았다.
