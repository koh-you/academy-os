# 시험분석 슬라이드 시퀀스 I5

## 기본 시퀀스

- 중학교: 표지 → 시험 구조 → 단원 비중 → 난이도 흐름 → 총평 → 주요문항/손풀이 반복 → 다음 대비 → CTA
- 고등학교: 중학교 흐름에 `점수 갈림 포인트`를 추가
- 표지, 시험 구조, CTA는 필수 역할이며 체크 해제할 수 없다.
- 나머지 역할은 교사가 제외할 수 있다.

## 조건부 생성

- 단원 비중, 난이도, 총평, 다음 대비는 해당 입력이 있을 때만 생성한다.
- 주요문항은 번호/제목/선정 이유/전략/오답 중 하나가 있는 블록만 생성한다.
- 손풀이 슬라이드는 풀이 전략 또는 검증 손풀이 자산 ID가 있을 때만 생성한다.
- 포함된 슬라이드는 조건부 제외 후 1번부터 연속 번호를 다시 부여한다.

## 누락 표시

슬라이드 자체를 만들 수 있지만 필수 입력이 부족하면 `needs_input`으로 유지한다. 예를 들어 문제 crop ID 없는 주요문항, 학교 스타일 없는 표지, 배점 구조 없는 시험 구조는 생성 목록에서 사라지지 않고 필요한 필드를 보여준다.

## 구현 경계

시퀀스 조립은 순수 함수이며 저장·AI·이미지 생성 side effect가 없다. 시퀀스 선택은 I3 작업본에 저장되고, 프롬프트 문자열은 I6에서 생성한다.
