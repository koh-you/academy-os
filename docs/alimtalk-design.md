# 알림톡 설계

목표: `koh_you_math` 웹앱에서 출결, 학부모 알림톡, 학생 알림톡을 Solapi 카카오 알림톡으로 발송한다.

## 명칭

- 데일리 리포트 알림톡 -> 학부모 알림톡
- 학생별 코멘트 알림톡 -> 학생 알림톡

코드 환경변수는 기존 배포 안정성을 위해 아래 이름을 유지한다.

```env
SOLAPI_DAILY_REPORT_TEMPLATE_ID=
SOLAPI_STUDENT_COMMENT_TEMPLATE_ID=
```

화면과 운영 문구에서는 `학부모 알림톡`, `학생 알림톡`으로 표시한다.

## 공통 원칙

- 실제 발송 전에는 `ALIMTALK_DRY_RUN=true`로 payload를 확인한다.
- 테스트 단계에서는 모든 수신자를 `01057882748`로 리다이렉트한다.
- 알림톡 본문은 큰 변수 하나에 조건부 문장을 조립한다.
- 값이 없는 항목은 문장 자체를 만들지 않는다.
- 카카오 알림톡 일반 템플릿은 자유로운 볼드/형광펜 처리가 어렵기 때문에 `[중요]` 텍스트로 강조한다.

## 출결 알림톡

템플릿명:

```text
출결 안내
```

변수:

```text
#{학원명}
#{학생명}
#{출결본문}
```

본문:

```text
[#{학원명}] 출결 안내

#{학생명} 학생 출결 안내입니다.

#{출결본문}
```

등원/하원은 지각분과 사유를 표시하지 않는다.

예시:

```text
2026-06-17 18:54에 등원 처리되었습니다.
수업: 월수금 7-10반
```

지각/결석/인정결석은 필요한 경우에만 사유를 표시한다.

예시:

```text
2026-06-17 19:18에 지각 처리되었습니다.
수업: 월수금 7-10반
지각: 18분
사유: 학교 수행평가 후 이동
```

## 학부모 알림톡

템플릿명:

```text
학부모 알림톡
```

기존 코드 환경변수:

```text
SOLAPI_DAILY_REPORT_TEMPLATE_ID
```

발송 예약 기준:

- 평일 수업: 기본 `22:30`
- 토요일 수업: 기본 `16:30`
- 수업 마무리, 코멘트 정리, 검수 때문에 기본 시각을 맞추기 어려운 경우 `30분 지연`을 선택한다.
- 30분 지연 시 평일은 `23:00`, 토요일은 `17:00`으로 예약한다.
- 강사가 누르는 순간 본문은 확정하고, 발송시각만 예약한다.
- 핵심 데이터가 비어 있으면 발송 15분 전 강사 알림 대상이 된다.

변수:

```text
#{학원명}
#{학생명}
#{수업일}
#{리포트본문}
```

본문:

```text
[#{학원명}] 학부모 알림톡

#{학생명} 학생의 #{수업일} 수업 안내입니다.

#{리포트본문}
```

리포트본문에 들어갈 수 있는 항목:

```text
출결
강의 교재
강의 내용
지난 숙제
다음 숙제
과제 상태
미완료 숙제
재시험 일정
보충 일정
코멘트
```

예시:

```text
출결: 출석
강의 교재: 쎈 공통수학1
강의 내용: 경우의 수
지난 숙제: GRIP ~987
다음 숙제: 쎈 643~647
과제 상태: 성실 완료
[중요] 재시험 일정: 6월 19일 금요일 공수1 누적테스트
코멘트: 경우의 수 조건 해석을 보완하면 좋겠습니다.
```

## 학생 알림톡

템플릿명:

```text
학생 알림톡
```

기존 코드 환경변수:

```text
SOLAPI_STUDENT_COMMENT_TEMPLATE_ID
```

변수:

```text
#{학원명}
#{학생명}
#{수업명}
#{수업일}
#{코멘트}
```

본문:

```text
[#{학원명}] 학생 알림톡

#{학생명} 학생에게 보내는 안내입니다.

수업일: #{수업일}
수업: #{수업명}

#{코멘트}
```

예시:

```text
[중요] 오늘 재시험 일정이 있습니다.
내용: 공통수학1 누적테스트
일시: 2026-06-17 18:00
수업: 월수금 7-10반
```

## 과제 상태 용어

추천 용어: `과제 상태`

이유:

- 단순 점수나 등급보다 넓다.
- 완료율, 난이도 적합성, 풀이 신뢰도, 성실도를 함께 기록할 수 있다.
- 학생/학부모에게 보여줄 때도 과하게 평가적으로 들리지 않는다.

초기 선택지:

```text
성실 완료
쉬움
80% 완료
아는 것만 풂
난이도 과함
풀이 확인 필요
미완료
미검사
```
