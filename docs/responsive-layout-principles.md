# 반응형 레이아웃 기준

academy-os의 역할별 화면은 같은 컴포넌트를 재사용하더라도 우선 화면 크기를 다르게 잡는다.

## 학생 화면

- 태블릿 화면을 1순위로 설계한다.
- 768px~1100px 구간에서는 요약 카드 3개와 상단 탭이 한 줄로 유지되도록 한다.
- 모바일은 2순위이며, 좁은 화면에서는 카드와 탭을 세로 흐름으로 접는다.

## 학부모 화면

- 모바일 화면을 1순위로 설계한다.
- 학부모는 알림, 리포트, 숙제, 출결을 빠르게 확인하는 목적이 크므로 단일 세로 흐름을 기본으로 한다.
- 큰 화면에서는 폭만 자연스럽게 확장하고, 모바일에서 읽히는 순서를 우선한다.

## 강사 화면

- 데스크톱과 노트북 화면을 1순위로 설계한다.
- 수업일지, 학생관리, 오답관리처럼 반복 입력과 표 조작이 많은 화면은 정보 밀도와 키보드 조작 효율을 우선한다.
