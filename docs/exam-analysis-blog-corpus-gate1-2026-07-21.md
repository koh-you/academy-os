- 전체 893건의 목록 썸네일 파일명에서 `ChatGPT` 문자열을 대조했다.

## 제외 기준

다음은 전체 893건에는 포함됐지만 시험별 슬라이드 프롬프트 코퍼스에서는 제외했다.

- 시험 대비 개강·특강·진도 현황만 다루는 글
- 성적 결과·학생 후기·입시설명회·학원 공지
- 특정 시험의 슬라이드 세트가 아닌 일반 학습 정보
- 같은 시험 슬라이드를 다시 묶은 종합글
- 단일 홍보 이미지 또는 외부 PDF 연결만 있는 글

제목에 `기말고사`, `중간고사`, `분석`이 들어가도 위 조건이면 자동 포함하지 않았다.

## Gate 2 입력 계약

Gate 2는 1차 코퍼스 21건을 기준으로 진행한다.

1. 기존 Vision 12건·116장은 raw 결과와 `logNo`를 먼저 대조하고 재호출하지 않는다.
2. 신규 9건·87장은 원본 URL, 파일명, 크기, 해시, 순서부터 수집한다.
3. 파생 종합글 19장은 원본 코퍼스와 해시를 대조해 중복을 제거한다.
4. `legacy_unconfirmed` 25건은 대표 표본 비교 전 유료 분석하지 않는다.
5. 유료 Vision이 필요하면 신규 호출 수와 예상 범위를 사람에게 먼저 보고한다.
