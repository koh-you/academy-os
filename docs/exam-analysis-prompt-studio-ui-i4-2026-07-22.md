# 시험분석 프롬프트 제작실 UI I4

## 위치와 충돌 회피

시험분석의 `최종 미리보기` 아래에 `GPT Image 슬라이드 프롬프트 제작실`을 표시한다. 병렬 리팩터링 세션이 수정 중인 `App.jsx`와 `App.css`는 건드리지 않고, 독립 컴포넌트와 독립 CSS를 `ExamAnalysisFinalPreviewPanel`에서 마운트했다.

## 입력 구조

- 학교급: 중학교/고등학교 전환. 학교명·학년으로 최초 제안하고 교사가 변경 가능
- 공통: 학교명, 학년, 시험명, 과목, 학교 스타일
- 시험 분석: 문항 수, 배점/문항 구조, 범위, 단원 비중, 난이도 근거, 총평
- 주요문항: 번호, 제목, 선정 이유, 개념, 전략, 오답 지점, 문제 crop ID, 검증 손풀이 ID
- 다음 대비: 한 줄에 하나씩 3~5개 학습 행동
- CTA: 학원 가치 문장, 검수된 연락/다음 행동

## 문구 드롭다운

I1의 21건·202장 기반 재서술 문구만 학교급·필드로 필터한다. 드롭다운 선택만으로 본문을 바꾸지 않고 `선택 문구 적용/이어 붙이기` 버튼을 눌러야 local draft에 반영한다. 적용 후 자유롭게 수정할 수 있다. 학교명, 문항 수, 배점, 범위, 자산 ID, 연락처 등 사실 필드에는 문구 옵션이 없다.

## 저장 표시

현재 패널 안에서 `편집 중 · 저장 필요 -> 저장 중 -> 서버 반영 확인 중 -> 저장 완료 · Supabase 재조회 확인` 또는 `저장 실패 · 작업본 유지`를 표시한다. 실패 시 입력값은 유지한다.
