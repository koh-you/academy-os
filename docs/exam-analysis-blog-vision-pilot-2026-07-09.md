# 시험분석 블로그 Vision AI 파일럿 - 2026-07-09

## 목적

네이버 블로그 시험분석 카드뉴스 이미지를 Vision AI로 읽어, 각 학교 시험의 실제 특징을 Academy OS의 블로그/인스타/카드뉴스 체크리스트와 키워드 사전으로 분류할 수 있는지 검증했다.

핵심 기준은 `이미지에 실제로 보이는 문구`와 `AI가 그 문구를 바탕으로 분류한 시험 특징`을 분리하는 것이다. 부교재, 모의고사, 시험범위, 문항번호, 난도 흐름은 보이는 근거가 있을 때만 확정 키워드로 둔다.

## 중단 조건

- 이미지에 보이지 않는 부교재/모의고사/문항번호를 AI가 확정처럼 말하면 중단한다.
- `visible_image` 근거와 `inferred_from_visible` 추론이 섞이면 중단한다.
- 학교별 실제 특징이 아니라 일반적인 공부 조언만 반복되면 프롬프트를 다시 줄인다.
- 벤치마킹 글의 문장을 장문으로 복제하는 결과가 나오면 사용하지 않는다.
